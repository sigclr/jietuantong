# 接团通（jietuantong）

面向中小地接社的接团与结算管理 SaaS 原型与产品文档。

## 目录

- [`web/`](web/) — React + Vite 可点击原型（新疆地接社场景）
- [`prototype/`](prototype/) — 早期 HTML 静态原型（备份）
- [`.cursor/plans/`](.cursor/plans/) — PRD 与实施计划

## 快速开始（React 原型）

```bash
cd web
npm install
npm run dev
```
